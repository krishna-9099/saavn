# KrishnaTune 🎧

KrishnaTune is a modern Android music streaming app built using Kotlin, Jetpack Compose, and Media3.

## 🚀 Features

- 🎵 Modern UI with Jetpack Compose
- 📱 Home screen with trending songs
- 🎧 Player screen with controls
- 🔽 Mini player for quick access
- 🧪 Mock data support (UI-first development)

## 🏗️ Architecture

- Clean Architecture (Data, Domain, UI)
- MVVM pattern
- Modular structure for scalability

## 🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Media3 (ExoPlayer)
- Retrofit (planned)
- Room (planned)

## 📂 Project Structure

```
android/
 ├── data/
 ├── domain/
 ├── ui/
 ├── player/
```

## 🧭 Current Status

UI development in progress (mock data).

## 🔮 Next Steps

- API integration (Saavn)
- Player service (background playback)
- Offline support
- Advanced UI animations
